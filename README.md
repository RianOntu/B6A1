Answer:any টাইপ দিয়ে বুঝায়,যেকোনো টাইপ আসতে পারে। যখন আমরা রিটার্ন টাইপ বা টাইপ টা কি হতে পারে তা জানি না তখন unknown ইউস করা হয়। যখন কোনো ফাঙ্কশন কখনোই কিছু রিটার্ন করে না বা কোনো ইনফিনিটি লুপ চলে ,যা কোনো টাইপ ই রিটার্ন করে না তখন ওই ক্ষেত্রে never ইউস করতে হয়।
